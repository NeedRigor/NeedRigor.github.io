二、程序的注释

1）单行注释

//
注意：字符串内容中的两根反斜线是内容的一部分

2）多行注释

/* */
注意：字符串内容中的/和/是内容的一部分，不是注释

4）快捷键

添加注释：Ctrl+k+c
取消注释：Ctrl+k+u

三、使用变量

变量是内存变量的简称，用于存放数据。

1）声明变量

语法：数据类型 变量名=值
常见：

整数（int）
浮点数（float和double）
字符（char）
字符串（string）
布尔（bool）

2）变量的赋值

语法：变量名=值

四、使用常量

常量是程序中固定不变的数据。

1）宏常量

一般在main函数的上面声明，用大写命名。
语法：#define 常量名 值

2）const修饰的变量

在程序的任何地方都可以声明。
语法：const 数据类型 常量名=值

3）常量的特点

程序中不允许改变常量的值，否则编译的时候会报错。

五、标识符的命名

1）C++命名规则

C++规定给标识符（变量、常量、函数、结构体、类等）命名时，必须遵守以下规则。

  • 在名称中只能使用字母字符、数字和下划线；
  • 名称的第一个字符不能是数字；
  • 名称区分大写字符与小写字符；
  • 不能将C++关键字用作名称；
  • 以下划线和大写字母打头的名称被保留给编译器及其使用的资源使用，如果违反了这一规则，会导致行为的不确定性。

六、输入数据

程序输入数据的方式有多种。

  • 从控制台的界面中输入（网页、PC桌面程序、APP程序）；
  • 从文件中读取；
  • 从数据库中读取；
  • 从网络中读取。

1）用std::cin输入数据

语法：std::cin>>变量名;
注意：

  • a）布尔型变量的值在计算机内部用1（true）和0（false）存储；
  • b）程序中可以书写true和false，也可以书写1和0，其它值将强制转换成1；
  • c）用cin输入时可以填1和0，其它值也强制转换成1；
  • d）用cout输出时只显示1和0，不显示true和false。
  • 如果输入的数据与变量的数据类型不匹配，会导致行为的不确定性。
